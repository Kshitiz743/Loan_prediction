# Loan Approval Prediction System

A Machine Learning-based web application built using Flask that predicts whether a loan application is likely to be approved.

## Features

- Predicts loan approval using a Decision Tree model
- User-friendly web interface
- Built with Flask
- Clean and responsive UI

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

## Installation

```bash
git clone https://github.com/<your-username>/Loan-Prediction-System.git
cd Loan-Prediction-System
pip install -r requirements.txt
python app.py
```

Open:

```
http://127.0.0.1:5000
```
